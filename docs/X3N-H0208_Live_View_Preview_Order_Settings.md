## Scene 1: 8\*IPCs & 0\*AHD

1. Connect EasyCheck to MDVR
   ![Streamax Easycheck](https://cdn.jsdelivr.net/gh/maozuxiao/Image-shack/Streamax%20Easycheck.gif)

2. Navigate to *==Preferences >> Preferences >> IPC Setup==* 

3. Configure the IP of the IPCs
   ![IPC Setup-1](https://cdn.jsdelivr.net/gh/maozuxiao/Misaki/image-20251215144934392.png)

4. Navigate to *==Preferences >> Preferences >> Live View >> Preview==* 

5. Configure the Startup Screen Preview Order & Check `No recording channel hidden`

   ![Live View >> Preview -1](https://cdn.jsdelivr.net/gh/maozuxiao/Misaki/image-20251215145306696.png)
