                      Introduction

                      Tip

                      This guide can help you automatically assign licenses to users in a specified group.

                       

                      Pre-configuration

                      Note

                      If you have completed Pre-configuration,Jump to Assign Team Plan for Group

                      B. Name & Department & Group, not required fields, will not be validated when submitting;

                      Add User Group

                      Group Name Enter the name of your User Group.

                      Parent Group Select the parent group from the dropdown, which can be left blank.

                      Group Administrator Choose the administrator for the User Group. Only team members with the Owner, Admin, or Group Administrator roles can be selected.

                      Caution

                      The hierarchy of User Groups can have a maximum of 5 levels. Levels exceeding 5 cannot be successfully set.

                      Assign Team Plan for Group

                      1. Navigate to Team Plan, select the account you need to manage, and click Assign Users

                      2. Check the Groups you want to assign the Team plan to, and click Submit

                      Expected Result: When the administrator adds users to the selected Groups, the Account Center will automatically assign the Team Plan to the added users.

                      Advanced Setting: Set Quota

                      1. Navigate to Team Plan, select the account you need to manage, and click Set Quota

                      2. Set quotas for different groups

                        Caution

                        The total quota should not exceed the number of Licenses

                      Expected Result: Users added or synchronized to the groups will automatically receive licenses on afirst-come, first-served basis. Once the license limit is reached, subsequent users will no longer be granted licenses.
