# 出差申请流程



## 提交出差申请单

1. 登录[锐明OA平台](http://oa.streamax.com:8080/sys/portal/page.jsp)
2. 访问常用新建/发起>>[费控报销](https://console-tc.huilianyi.com/main/dashboard)
3. 找到`快捷提单`>>`提单据`>>`申请单`（`出差申请单`）
4. 填写出差申请单的`单据信息`内容，示例如下
   1. *核算体系`深圳市锐明技术股份有限公司`*
   2. *费用承担部门`深圳市锐明技术股份有限公司|海外营销中心|欧洲战区|土耳其区`*
   3. *参与人员（含随行人员）`毛祖濂`*
   4. *是否合住`是`*
   5. *开始日期-结束日期`2025-08-06 至 2025-10-16`*
   6. *是否购买境外保险`是`*
   7. *申请事由`前往土耳其区域，进行公司业务的现场支持。`*

5. 申请明细中点击`新建行程` ，参考如下填写`申请明细内容`后点击`保存`
   1. 选择行程类型：*单程`往返`多程*
   2. *出发地（去程）`香港`*
      *目的地（去程）`伊斯坦布尔`*
      *开始日期`2025-08-06`*
      *结束日期（去程对应返程开始）`2025-10-16`* 
      *出发地（返程）`伊斯坦布尔`*
      *目的地（返程）`香港`*
      *结束日期（返程）`2025-10-16`* 
      *事由`前往土耳其区域，进行公司业务的现场支持。`* 

6. 点击`计算差补`>>`确认`>>`展开`，确认数值无误后，点击`保存`
7. 点击行程中的`新建明细`，增加`机票` `签证` `交通` `酒店`等费用 (第二程中，如果没有费用，可能无法提交)
8. 点击`提交`
